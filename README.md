## Project Overview
This project implements a Production-grade **Agentic Retrieval-Augmented Generation (RAG)** system designed to ingest, process, and query complex legal and financial documents (The Epstein Files). 

Unlike standard linear RAG pipelines, this system utilizes **LangGraph** to create a multi-step, cyclic reasoning agent. The agent dynamically evaluates the relevance of retrieved documents, hallucinates less, and securely manages context using a **FAISS** vector index. The backend is powered by **Django** and secured via **JSON Web Token (JWT)** authentication.

## System Architecture
The application architecture follows a strict separation of concerns:
1. **Frontend:** Vanilla JavaScript integrating HTTP Bearer Token authentication.
2. **Backend API:** Django-based REST endpoints handling user sessions and JWT issuance.
3. **Graph Engine (AI Core):** LangGraph state machine orchestrating the `gpt-4o-mini` LLM.
4. **Vector Storage:** Local FAISS index holding high-dimensional embeddings generated by `BAAI/bge-large-en-v1.5`.
